# Retail Store Customer Clustering

Every dataset contains structure that nobody designed. The work is learning to see it.

## Project Overview

A segmentation project exploring hidden structure in retail customer data.

Applied K-Means clustering to surface natural customer groupings and used PCA to visualise patterns in a way that made them interpretable.
